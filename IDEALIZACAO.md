# IDEALIZAÇÃO - Gerador de Memes

## Qual problema real este projeto resolve?

Este projeto resolve a necessidade de criar memes personalizados de forma rápida e simples, sem precisar de ferramentas complexas de edição de imagem. É uma solução prática para:

- Aprender manipulação de Canvas HTML5
- Entender como trabalhar com imagens e texto em JavaScript
- Criar uma aplicação web interativa e visual
- Praticar integração de múltiplas tecnologias (HTML, CSS, JavaScript)

## Quem é o usuário final e o que ele precisa?

**Usuário Final:** Pessoas que querem criar memes personalizados de forma rápida e divertida.

**Necessidades:**
- Interface simples e intuitiva
- Preview em tempo real do meme
- Múltiplas opções de imagens de fundo
- Download fácil do meme gerado
- Funcionamento em desktop e mobile

## Casos de uso principais

1. **Escolher imagem de fundo:** Usuário seleciona uma imagem da lista disponível
2. **Digitar textos:** Usuário insere texto superior e inferior nos campos
3. **Ver preview:** Usuário visualiza o meme sendo gerado em tempo real
4. **Baixar meme:** Usuário clica em download e recebe um arquivo PNG

## Experiência do usuário

**Princípios:**
- **Simplicidade:** Interface limpa, sem elementos desnecessários
- **Feedback visual:** Preview atualiza automaticamente ao digitar
- **Responsividade:** Funciona bem em desktop e mobile
- **Legibilidade:** Textos com contorno preto para ficarem visíveis em qualquer imagem
- **Acessibilidade:** Campos grandes, botões claros, feedback imediato

## Tecnologias utilizadas

- HTML5 (estrutura)
- CSS3 (estilização responsiva)
- JavaScript (lógica e manipulação de Canvas)
- Canvas API (renderização de imagem e texto)
- Git/GitHub (versionamento e deploy)
